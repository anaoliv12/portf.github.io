HTML5

HTML5 é a última versão do padrão que define o HTML, a linguagem de marcação que estrutura o conteúdo na web. Ele introduz e aprimora diversos recursos, incluindo suporte a áudio e vídeo, APIs para interação com o navegador e melhorias na semântica e acessibilidade.

JAVASCRIPTY

Java é uma linguagem de programação de alto nível, orientada a objetos e independente de plataforma. Ela é conhecida por sua portabilidade, segurança e capacidade de desenvolver aplicativos robustos para diversas finalidades, desde aplicações web até aplicativos móveis.

CSS

CSS (Cascading Style Sheets) é uma linguagem de estilo usada para controlar a apresentação visual de documentos HTML. Ele permite a personalização de layouts, cores, fontes e outros aspectos do design de uma página web. O CSS é fundamental para criar interfaces atraentes e responsivas.
